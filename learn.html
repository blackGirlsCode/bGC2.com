What is a front-end developer?

Architechts and develops websites and applications using web technologies to run on a web platform. Most technologies ussed by these professionals are HTML, CSS, and Javascript

How are Front-End Deveopers Made?

Nyone with the right determination can teach themselves to be a front-end developer. This is one profession that is practically free for the taking with an internet connectuin and a deep desire for knowledge, regardless of prior academic performance.

What should I learn?

Learn the underlying technologies rather than abstractions. Steer from tools that hide compleity to avoid having the appearance of advanced skills, while not having the understanding of basic fundamentals.
